WYSIWYG HTML Editor

WYSIWYG HTML Editor can be realized using QtWebKit, by utilizing the contentEditable property of a web page. If this is set to true, "live editing" of the web page is possible.

If you are viewing this document in the HTML Editor demo, you can edit this HTML file to explore QtWebKit editing features. We have included some comments in each of the following sections to encourage you to experiment.

Formatting

This editor supports bold, italic, underline, and strikethrough character formatting. If you place the cursor in a region of formatted text, the controls in the tool bars will change to reflect the current formatting.

The text can use different font, like Helvetica or Times. Even with different colors, and can be highlighted as well. Different font sizes are supported: xx-small, x-small, small, medium, x-large, and xx-large.

Paragraphs can be indented, for example like this one. In HTML term, this is known as "blockquote".

Paragraphs can be formatted so that the text is left-aligned, right-aligned, centered, or fully justified.

This paragraph is centered, ...

while this one is right-aligned, ...

Try changing the alignment of some text and resize the editor to see how the text layout changes.

Each paragraph can be set as a heading. Use menu Format, Style and choose a proper heading, Heading 1 is for the highest-level. A heading can be reverted back to a normal paragraph by choosing Paragraph in that Style menu.

Lists

Both bulleted and numbered lists are supported.

Here is an example of bulleted list:

  • Mercurius
  • Venus
  • Earth

And this is an example of numbered list:

  1. Mars
  2. Jupiter
  3. Uranus
  4. Saturnus

Hyperlinks

Since this is an HTML editor, it is not complete without the support for create hyperlinks. You can do this by using the icon in the tool bar or menu Edit , Create Link. Here is an example of a link to Qt Software website.

If a link is clicked, it will be opened in the default web browser. A confirmation dialog pops up before the link is opened.

Undo and Redo

Full support for undo and redo operations is built into the editing feature. Try making changes to this document and press Ctrl+Z to undo them. You can always recover the original contents of the document.
